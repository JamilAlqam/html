الأوسمة القديمة غير المدعومة في HTML5

هذه الأوسمة تم إهمالها في HTML5 ولا ينصح باستخدامها:

  • <acronym> - يستخدم <abbr> بدلاً منه
  • <applet> - يستخدم <embed> أو <object> بدلاً منه
  • <basefont> - يتم التحكم بالخطوط عبر CSS
  • <big> - يستخدم CSS بدلاً منه
  • <center> - يستخدم CSS (text-align: center) بدلاً منه
  • <dir> - يستخدم <ul> بدلاً منه
  • <font> - يستخدم CSS بدلاً منه
  • <frame>, <frameset>, <noframes> - تم استبدالها بتقنيات أخرى
  • <strike> - يستخدم <del> أو <s> بدلاً منه
  • <tt> - يستخدم CSS بدلاً منه
